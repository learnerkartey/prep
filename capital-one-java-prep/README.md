# Capital One Java Prep

GitHub-ready Java interview prep repo built around the problems you selected for Capital One / CodeSignal-style practice.

## Structure

- `INDEX.md` - master navigation page
- `easy/`, `medium/`, `hard/` - markdown notes for each problem
- `src/easy/`, `src/medium/`, `src/hard/` - Java solutions
- `categories/` - quick category navigation pages
- `TEMPLATE.md` - file format used for each problem note

## Counts

- Easy: 12
- Medium: 37
- Hard: 11
- Total: 60

## Suggested study order

1. Finish all Easy problems for speed and accuracy.
2. Focus on Medium problems in these categories:
   - Arrays / HashMap
   - Sliding Window
   - Matrix / Grid
   - Intervals
   - Design
3. Use Hard problems for pattern recognition and optimization rounds.

## CodeSignal-oriented tags

- Warm-up: quick implementation or direct simulation
- Matrix / Grid: likely Q3-style practice
- Optimization: likely Q4-style practice
- Design: class design and state management
